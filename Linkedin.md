# 💼 Best Practices for LinkedIn Posts

LinkedIn favors professional storytelling, value-driven content, and readability.
*   **The "Hook"**: Start with a personal insight or a professional problem statement.
*   **Structure**: Use short paragraphs and lists. Wall of text = skip.
*   **Media**: Native video uploads perform best, followed by image carousels or single high-quality screenshots.
*   **Engagement**: Ask a question at the end to encourage comments.

## 🚀 The "Project Launch" Post

**Headline:** I built a privacy-first AI tool to solve digital clutter. Here’s why and how. 👇

We all have those folders full of files named `document_final_v2.pdf` that we're afraid to delete. Traditional disk cleaners help with space, but they don't help with *context*.

That's why I created **CleanIQ**.

It's a smart file manager that leverages **Local AI** to understand your documents. Instead of just showing you "100MB used," it reads the file locally and gives you a 1-sentence summary and an importance rating.

**💡 The Core Philosophy: Privacy First**
In the age of AI, uploading personal docs to the cloud for analysis is a risk. I built this tool to run 100% locally using **Ollama** and local LLMs (like Llama 3 or Mistral). No data ever leaves your machine.

**🛠️ The Tech Stack:**
*   **Python & FastAPI** for a robust backend
*   **Vanilla JS & CSS** for a clean, dependency-free UI
*   **Ollama Integration** for powerful, offline AI analysis

**✨ Key Features:**
✅ **Smart Scanning:** Instantly identifies large, forgotten files.
✅ **AI Summaries:** Understand file contents without opening them.
✅ **Apple-Inspired Design:** A UI that feels premium and native.

I’m releasing this as an open-source project for anyone who wants to reclaim their storage intelligently and privately.
